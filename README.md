🚀 Возможности
Управление компонентами: Полный CRUD (создание, чтение, обновление, удаление) электронных компонентов

Категоризация: Автоматическая группировка по типам компонентов (микроконтроллеры, транзисторы, диоды и т.д.)

Управление даташитами: Прикрепление PDF-документации к компонентам

Складской учет: Отслеживание количества компонентов и их расположения

Автоматическое журналирование: Фиксация даты и времени всех изменений

Гибкая настройка: Возможность изменения путей хранения файлов через конфигурацию

📋 Требования
Операционная система: Windows 10/11

.NET Runtime: .NET 8.0

База данных: MS SQL Server LocalDB

Для просмотра даташитов: Любое приложение для просмотра PDF-файлов

🛠️ Установка и запуск
Автоматическая установка (рекомендуется)
Скачайте последнюю версию приложения из раздела Releases

Распакуйте архив в выбранную папку

Запустите ElectronicComponentStore.exe

При первом запуске:

Автоматически создается база данных ElectronicComponentsDB

Создается таблица Components с тестовыми данными

Создается папка для даташитов (C:\ElectronicComponents\Datasheets)

Ручная сборка из исходного кода
bash
# Клонирование репозитория
git clone https://github.com/your-username/electronic-component-store.git
cd electronic-component-store

# Сборка проекта
dotnet build
dotnet run
📖 Инструкция по использованию
Главный экран
https://via.placeholder.com/800x500?text=Main+Application+Screen

Главное окно разделено на три основные области:

Левая панель - категории компонентов

Центральная область - таблица с компонентами

Нижняя панель - панель инструментов

📥 Добавление новых компонентов
Способ 1: Создание нового компонента с даташитом
Нажмите кнопку "Добавить" в нижней панели

Заполните форму в диалоговом окне:

Название * (обязательно) - имя компонента

Тип * (обязательно) - выберите из списка или введите новый

Номер ячейки - место хранения на складе

Количество - текущий запас

Даташит - нажмите "Выбрать файл" для прикрепления PDF-документации

Нажмите "OK" для сохранения

Способ 2: Добавление даташита к существующему компоненту
Выберите компонент в таблице

Нажмите кнопку "Редактировать"

В диалоговом окне нажмите "Выбрать файл"

Выберите PDF-файл в диалоговом окне

Файл автоматически сохранится - имя файла появится в поле "Даташит"

Нажмите "OK" для завершения редактирования

🔍 Просмотр и поиск компонентов
Фильтрация по категориям
Нажмите на название категории в левой панели для показа компонентов только этого типа

Нажмите "Все компоненты" для сброса фильтра

Просмотр деталей компонента
Дважды щелкните по названию компонента в таблице

Отобразится подробная информация только об этом компоненте

📄 Работа с даташитами
Открытие даташитов
В таблице компонентов найдите столбец "Даташит"

Нажмите на подсвеченное синим название файла

Файл откроется в программе по умолчанию для PDF-файлов

Требования для открытия даташитов
На компьютере должна быть установлена программа для просмотра PDF

Файл должен существовать в указанной папке даташитов

✏️ Редактирование компонентов
Выберите компонент в таблице

Нажмите кнопку "Редактировать"

Внесите необходимые изменения

Нажмите "OK" для сохранения

Примечание: Поле "Дата изменения" обновляется автоматически.

🗑️ Удаление компонентов
Выберите компонент в таблице

Нажмите кнопку "Удалить"

Подтвердите удаление в диалоговом окне

📁 Управление папкой даташитов
Изменение расположения папки
Нажмите кнопку "Изменить папку даташитов"

В диалоговом окне выберите новую папку

Нажмите "OK"

Приложение автоматически:

Обновит настройки

Создаст новую папку, если необходимо

Покажет подтверждающее сообщение

Quantity	    INT	           Количество в наличии
DateOfChanges	DATETIME2	   Дата последнего изменения
Datasheet	    NVARCHAR(255)  Имя файла даташита

⚙️ Настройки приложения
Файл конфигурации appsettings.json
json
{
  "ConnectionStrings": {
    "DefaultConnection": "Data Source=(localdb)\\MSSQLLocalDB;Initial Catalog=ElectronicComponentsDB;Integrated Security=True"
  },
  "AppSettings": {
    "DatasheetsFolder": "C:\\ElectronicComponents\\Datasheets",
    "ApplicationName": "Магазин электронных компонентов"
  }
}
Ручное изменение настроек
Вы можете отредактировать файл appsettings.json в папке приложения:

json
{
  "AppSettings": {
    "DatasheetsFolder": "D:\\Ваша\\Папка\\Datasheets"
  }
}

❗ Устранение неполадок
Проблема: Даташит не открывается
Решение:

Проверьте, установлена ли программа для просмотра PDF

Убедитесь, что файл существует в папке даташитов

Попробуйте открыть файл напрямую через Проводник

Проблема: Не удается изменить папку даташитов
Решение:

Проверьте права доступа к выбранной папке

Убедитесь, что папка не заблокирована антивирусом

Попробуйте выбрать другую папку

Проблема: Компоненты не сохраняются
Решение:

Проверьте заполнение обязательных полей (Название, Тип)

Убедитесь, что база данных работает корректно

Проверьте подключение к LocalDB

Проблема: Приложение не запускается
Решение:

Убедитесь, что установлен .NET 8.0 Runtime

Проверьте наличие MS SQL Server LocalDB

Посмотрите логи в папке приложения
